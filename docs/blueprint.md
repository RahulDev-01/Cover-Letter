# **App Name**: Cover Letter Pro

## Core Features:

- Input Fields: Provide input fields for personal information (name, contact), recipient information (company, contact), and job details.
- AI-Powered Generation: Generate a personalized cover letter using AI, highlighting key achievements and skills tailored to the job description, with a tool for suggesting specific examples to showcase abilities.
- ATS Keyword Optimization: Analyze the job description and optimize the generated cover letter for Applicant Tracking Systems (ATS) by identifying and incorporating relevant keywords.
- Template Selection: Offer a selection of cover letter templates, with one closely resembling the layout provided in the request (sender details, recipient details, body, closing).
- Real-time Preview: Display a real-time preview of the cover letter as the user fills in the input fields and as the AI generates content.
- Download: Allow users to download the generated cover letter in PDF format.

## Style Guidelines:

- Background color: Soft, desaturated beige (#F5F5DC) to provide a professional and clean canvas, similar to the reference image.
- Primary color: Deep brown (#4E342E) for headings and key elements, ensuring readability and a touch of sophistication.
- Accent color: Muted gold (#BDB76B) to highlight important information and call-to-action buttons, adding subtle visual interest.
- Body text: 'Literata', serif, to mirror the original document and support readability.
- Headlines: 'Space Grotesk', sans-serif, to pair with 'Literata' and provide a balance of formality and modernity.
- Maintain a clean and structured layout with ample whitespace, mirroring the provided image.
- Use subtle dividers (thin lines) to separate sections, emulating the reference document's organized design.
